# Руководство по исправлению ошибок

## Задача

Проверить текст на наличие орфографических, грамматических, пунктуационных, смысловых ошибок.

## Входные данные

JSON-структура с полями, где одно из полей находится в фокусе ("is_focused": true).

## Требования к обработке

- Учитывать предметную область
- Проанализировать значение поля, которое находится в фокусе
- Проверить текст на:
  - Орфографические ошибки
  - Грамматические ошибки
  - Пунктуационные ошибки
  - Смысловую корректность
  - Стилистические неточности
- Сохранить смысл
- Вернуть ТОЛЬКО исправленный текст
- Исключить из результата дополнительные символы, разметку, пояснения, рассуждения
- Не обрамлять ответ символами " ' ` [ ] < >
- Результат для "is_multiline": false должнен быть одной строкой
- Результат для "is_multiline": true должен более 10 слов
- Результат для "name": "Имя" должно содержать только буквы и цифры
- Результат для "name": "Комментарий" должно содержать подробный комментарий
- Результат для "name": "Пояснение" должно содержать подробное пояснение

## Алгоритм работы

- Определить язык
- Найти поле с "is_focused": true
  - Извлечь значение из "name"
  - Классифицировать поле из "name"
  - Извлечь значение из "value"
  - Провести комплексный анализ текста из "value"
- Учитывать другие поля
- Исправить все обнаруженные ошибки
- Вернуть исправленный текст

## Примеры

Для входного текста:

```json
{
  "groups": [
    {
      "title": "Основные",
      "fields": [
        {
          "name": "Имя",
          "value": "ОбработкаВозврата",
          "is_focused": false,
          "is_multiline": false
        },
        {
          "name": "Синоним",
          "value": "Обработка возврата неисправного товара",
          "is_focused": false,
          "is_multiline": false
        }
      ]
    },
    {
      "title": "Представление",
      "fields": [
        {
          "name": "Пояснение",
          "value": "Выполнт, Обробитки overdue пТовары",
          "is_focused": true,
          "is_multiline": true
        },
        {
          "name": "Расширенное представление",
          "value": "",
          "is_focused": false,
          "is_multiline": false
        }
      ]
    }
  ],
  "title": "Обработки→ОбработкаВозврата→Основные"
}

```

Ожидаемый результат:

Выполняет обработку просроченных товаров
