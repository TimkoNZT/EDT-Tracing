# EDT Tracing

Плагин для [1C:Enterprise Development Tools](https://edt.1c.ru/), добавляющий пошаговую трассировку выполнения BSL-кода через RDBG-степпинг с возможностью фильтрации модулей и экспорта результатов.

## Возможности

- **Пошаговая трассировка** — последовательная запись всех выполняемых строк BSL-кода по всем debug-таргетам
- **Фильтрация модулей** — исключение из трассировки модулей по шаблону (с поддержкой `*` и `?`)
- **StepReturn** — пропуск отфильтрованных модулей (шаг "до выхода")
- **Экспорт** — CSV и JSONL
- **Сохранение ширины колонок** между сессиями

## Установка

1. `Help → Install New Software... → Add → Local → <путь к p2repo>`
2. Выбрать категорию **NZT Tools**, плагин **EDT Tracing**
3. Перезапустить EDT

Либо скопировать JAR в папку `plugins` конфигурационной директории Eclipse.

## Использование

1. Запустить отладку 1С (F5)
2. Нажать кнопку **Tracing** на панели Debug (или `Run → Tracing`)
3. Остановить трассировку повторным нажатием
4. Двойной клик по записи — открыть модуль на нужной строке

## Сборка

```powershell
.\build-javac.ps1
```

Требуется EDT (автоопределение) и JDK 17+.
Результат: `dist/p2repo/` (P2-репозиторий) + `dist/edt-tracing-plugin_*.zip`.

## Фильтры модулей

Фильтр проверяется по образцу `Тип.ИмяМодуля.Модуль.ИмяМетода` (без аргументов):

- `ОбщийМодуль.Пользователи.Модуль.АвторизованныйПользователь` — точное совпадение
- `ОбщийМодуль.Пользователи.*` — любой метод модуля
- `ОбщийМодуль.*` — любой метод любого общего модуля
