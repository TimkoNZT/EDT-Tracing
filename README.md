# EDT Tracing

Плагин для [1C:EDT](https://edt.1c.ru/), добавляющий пошаговую трассировку выполнения BSL-кода с возможностью фильтрации модулей и экспорта результатов.

## Возможности

- **Пошаговая трассировка** — последовательная запись всех выполняемых строк BSL-кода по всем debug-таргетам
- **Фильтрация модулей** — исключение из трассировки модулей по шаблону
- **Экспорт** — CSV и JSONL

## Установка

- **Через GitHub:** Help → Install New Software → Add → `https://timkonzt.github.io/EDT-Tracing/p2repo/`
- **Из ZIP-архива:** Help → Install New Software → Add → Archive → указать `edt-tracing-plugin_<версия>.zip` из [релизов](https://github.com/TimkoNZT/EDT-Tracing/releases)



## Использование

1. Запустить отладку 1С (F5) **
2. Нажать кнопку **Tracing** на панели Debug (или `Run → Tracing`)
3. Остановить трассировку повторным нажатием или дождаться окончания выполнения
4. Двойной клик по записи — открыть модуль на нужной строке
* п.1 и п.2 можно выполнять в любом порядке

## Фильтры модулей

Фильтр проверяется по образцу `Тип.ИмяМодуля.Модуль.ИмяМетода` (без аргументов):

- `ОбщийМодуль.Пользователи.Модуль.АвторизованныйПользователь` — точное совпадение по имени метода
- `ОбщийМодуль.Пользователи.*` — любой метод модуля Пользователи
- `ОбщийМодуль.ОбщегоНазначения*` — любой из модулей ОбщегоНазначения...
